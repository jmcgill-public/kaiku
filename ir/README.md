# ir

Impulse responses. One room per composition, committed.

---

## provenance

Every IR file in this directory has a provenance entry below. Format:

```
filename:     the file as it lives here
source:       how it was captured or generated
character:    what kind of space it describes
used in:      which comp/ document references it
notes:        anything a future engineer needs to know
```

The room is committed per the 1982 ethic. It is not revisited once it is chosen.

---

## format

- WAV, 48kHz, 32-bit float, mono
- Length: sufficient tail (minimum 2x RT60)
- Named in the project vocabulary -- Finnish where a Finnish word exists for the space

## the rule

One room per track. Not one room per instrument, not one room per session -- one room per track, decided early and not changed.

If the room is wrong, the track is wrong. Choose before you commit.
