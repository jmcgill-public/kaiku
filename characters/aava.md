# Aava

Has not left the ship in seventeen attempts. The shuttle left without her.
All seventeen times. This is not a disorder. It is a decision, or something
prior to a decision.

Perceives the void directly, in ways the instruments can't replicate.
This is not mysticism. It is a different set of senses used correctly.

Is already at the destination the rest of the novel is trying to reach.
The waiting looks like patience. It isn't patience.

Will provoke a strong reaction. That's the correct response.
