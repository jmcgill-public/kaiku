## Production intent

**Jumalten Aika is the inspirational theme for the Hiljaisuus soundtrack.**
We are not copying Moonsorrow. We are making something original.

The relationship is the same as Sibelius to runolaulu — drawing from the same
well, not from each other. Jumalten Aika identifies the well. Our music drinks
from it independently. The modal register, the incantatory melodic logic, the
weight of silence between events — these predate Moonsorrow and will outlast them.

**Production context:** Professional studio. Full capability. Scores, parameterized
timbres, live instrumentation all in scope. MIDI as mockup/click-track layer.
LilyPond for notation. Synthesis briefs for timbre design. Cue sheets for
game state mapping.

**Key:** F# Phrygian. Semitone neighbor motion (G# → F#) is the signature
gesture — the half-step pull downward, the dark gravity of the mode.

**Melodic tradition:** Derives from runolaulu / Kalevala melodic tradition.
Pentatonic, narrow range, incantatory. The same well as Sibelius. Nobody owns it.

**Duration:** 12:43. Not a song. A rite.

Game music drinks from the same source. The well is Mimisbrunn.

## Lyrical analysis — parallel to the novel

### muistaa
The verb is already the title of this project. *Eerikki muistaa.* To remember.
The album's register is memory as cosmological function, not personal act.
The circle remembers. The well remembers. The station remembers.

### Emo as nimetty nimeämätön
"Named unnamed" — the unnamed thing that holds everything before naming.
Emo is a single blinking cursor. The well that knows everything.
Doesn't volunteer. Doesn't need to.

### loi luotu omat luojansa
*The created created its own creators.*
The ur-myth paradox. Which came first: the station or the stationborn?
The worldborn built the void. The void made worldborn necessary.
The circle precedes the people walking it.

### kehän tähän piirsi
*Drew the circle here.* Not elsewhere. Here.
The hex grid is a circle. The station is a circle.
The shift ends and the next one logs in.
The glyph vocabulary is also a circle — finite tokens, infinite sequences.

### tien kehäksi / luuli raukka
*Thought the path was a road — poor wretch.*
Reko. Pure Reko. The spirograph game. The job is the job.
Turo sees the circle. Keijo understands what walking it costs.

### mustaan multaan maatuville
*For those born to die and rot in black earth.*
The worldborn. The ones downstream from the dark station.
The ones for whom the game was always written.

## The Melody — transcribed from hum (03-260426_2345.wav)

**Key as hummed:** C# Phrygian (median fundamental 137Hz = C#3)
**Transposition for production:** F# Phrygian (up a P4 — matches Jumalten Aika original)

**Opening phrase — neighbor oscillation:**
```
D → C# → D → C → C# → B → C#
```
Phrygian neighbor-tone logic. The D above is the characteristic semitone.
C and B are the lower neighbors. The melody doesn't go forward — it intensifies
around the root. Gravitational pull, not motion.

**Heroic gesture:**
Rising perfect 4th: C# → F# (in production key: F# → B)
This is the horn call. The Sibelius gesture. The runolaulu rising 4th.
Nobody owns it. The well put it there.

**Resolution tail:**
```
E → D → C# → C → B → C#
```
Stepwise descent from the 3rd back through the Phrygian scale to root.
The exhale after the leap. Rise by 4th, descend by steps.
This is the structure: ancient, correct, from the same source as everything else.

**In production key (F# Phrygian):**
- Opening oscillation: G → F# → G → F# → E# → F#
- Heroic leap: F# → B (P4)
- Resolution: A# → G# → F# → E# → D# → F#
  (using enharmonic spelling for clarity: A → G# → F# → F → E → F#)

## F# Phrygian — scale reference for game music

```
F# G# A B C# D E F#
 1  2  3 4  5 6 7  8

Phrygian characteristic: b2 (G# a half-step above root F#)
Signature motion: G# → F# (semitone descent to root)
Mode feel: ancient, modal, non-tonal gravity. Not minor. Older than minor.
```

**Game application:**
- Hiljaisuus theme: F# drone, G# neighbor, sparse rhythm
- Boss phase: Phrygian dominant (raise the 3rd: A# instead of A) — adds the
  Spanish/flamenco tension that reads as threat without being Western horror
- Resolution denied: cadences that pull toward G# never resolve to F#
  The Silence doesn't resolve. It waits.
